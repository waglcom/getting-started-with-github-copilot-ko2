# GitHub Copilot 시작하기

_1시간 이내에 GitHub Copilot을 시작해 보세요._

## 환영합니다

- **누구를 위한 실습인가요**: 코드 작업 속도를 높이고 싶은 모든 수준의 개발자
- **무엇을 배우나요**: Copilot으로 코드 설명, 작성, 계획, 개발을 수행하는 다양한 상호작용 방식
- **무엇을 만들게 되나요**: Copilot을 활용해 Mergington High School의 방과후 활동 웹사이트를 업데이트합니다.
- **사전 준비사항**:
  - Skills 실습: [Introduction to GitHub](https://github.com/skills/introduction-to-github)
  - [VS Code](https://code.visualstudio.com/) 사용 경험
  - 기본적인 코딩 원리
- **소요 시간**: 이 실습은 1시간 이내에 완료할 수 있습니다.

이 실습에서 다음을 수행합니다.

1. 미리 구성된 Codespace를 사용해 브라우저에서 VS Code를 실행합니다.
1. GitHub Copilot으로 개발하고 계획하는 다양한 상호작용 옵션을 익힙니다.
1. Copilot을 사용해 Pull Request를 요약하고 리뷰합니다.

### 실습 시작 방법

아래 실습 리포지토리를 내 계정으로 복사한 뒤, Octocat(Mona)가 첫 번째 레슨을 준비할 수 있도록 **약 20초** 기다렸다가 **페이지를 새로고침**하세요.

> 🌍 **한글화 버전**: 이 리포지토리는 한국 개발자를 위해 완벽하게 한글화되었습니다. 아래 "실습 복사" 버튼을 클릭하면 한글 버전으로 자동 복사됩니다.
